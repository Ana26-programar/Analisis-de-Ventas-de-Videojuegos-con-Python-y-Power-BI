# Analisis-de-Ventas-de-Videojuegos-con-Python-y-Power-BI
Este proyecto analiza las ventas globales de videojuegos desde 1980 hasta 2017, con el objetivo de identificar tendencias temporales, plataformas dominantes, regiones clave y juegos de mayor impacto comercial.
